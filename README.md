# streamlit_autobio

## Introduction

This project is a Streamlit-based web application for displaying an autobiography and a gallery. It is designed for educational purposes and demonstrates how to use Streamlit to create interactive web apps in Python.

## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/JohnEdward-Selma/streamlit_autobio.git
   cd streamlit_autobio
   ```
2. (Recommended) Create and activate a virtual environment:
   ```sh
   python -m venv venv
   # On Windows:
   venv\Scripts\activate
   # On macOS/Linux:
   source venv/bin/activate
   ```
3. Install the required dependencies:
   ```sh
   pip install -r requirements.txt
   ```

You are now ready to run the Streamlit app!
